About This Seedling App

This app aims to create a cryptographically secure random number that is then SHA-256 encoded. This random token is named "seedling" in this app and is generated from true random data provided by the user via uploading a .csv file containing numerical data from physical measurements of a physical system. A great example of use of this app would be to upload the random data found in the .csv files obtained from measuring the efficiency of Proton Exchange Membrane Fuel Cells (a type of non-fossil fuel based energy source). Please refer to datasets from this kaggle website if you wish to practice retrieving random data. These Seedlings can be used as hashed cryptographic keys used in encryption, as passwords, as tokens, etc.
